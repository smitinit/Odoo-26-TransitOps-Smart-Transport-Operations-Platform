# TransitOps Frontend

Next.js frontend for **TransitOps** – Smart Transport Operations Platform, built for the Odoo Hackathon.

This app provides the UI for managing fleet, drivers, trips, maintenance, and operational expenses.

---

## Tech Stack

- Next.js
- Tailwind CSS

---

## Features

- Secure Authentication
- Dashboard with Fleet Overview
- Vehicle Management
- Driver Management
- Trip Management
- Maintenance Tracking
- Fuel & Expense Tracking
- Reports & Analytics

---

## Getting Started

### Prerequisites

- Node.js installed
- Backend running at `http://localhost:8000` (see root [README](../README.md))

### Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
